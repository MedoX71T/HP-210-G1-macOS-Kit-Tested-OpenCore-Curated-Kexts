# Changelog

All notable changes to this project will be documented in this file.

## [Unreleased]
- Small README SEO improvements: added Table of Contents, CONTRIBUTING, ISSUE templates, diagnostics script, and release checksums automation.

## [v5.3] - 2026-01-06
- Cumulative development through v5.3: release workflow, release asset checksums, README SEO and documentation improvements, diagnostics tooling, and CPU/GPU performance optimizations (power management and iGPU tuning).

## [v4.0] - 2025-10-05
- Major stability and cleanup release: refreshed kext stack (Lilu/WhateverGreen/Lilu-plugins), fixed USB mapping regressions, and improved battery/sleep reliability.
- Updated ACPI patches and consolidated SSDTs for more consistent sleep/hibernation behavior.

## [v3.0] - 2025-07-22
- Added Sonoma compatibility testing and initial EFI layout for macOS Sonoma (experimental). Improved trackpad and PS2 controller support.
- Audio layout refinements and microphone fixes for recording.

## [v2.0] - 2025-03-10
- OpenCore upgrade and refactor: config.plist cleaned and modularized, driver/ kext loading order optimized.
- Introduced automated release packaging and basic SHA256 checksum generation for release assets.

## [v1.1] - 2024-12-15
- Minor fixes and improvements: adjusted AppleALC layout settings, fixed brightness keys behavior, and updated Wifi/BT kexts for improved connectivity.
- Small ACPI tweaks to improve battery reporting and power management.

## [v1.0.0] - 2024-11-02
- Initial release: OpenCore 1.0.5, kext updates, ACPI patches, and base EFI structure. Supported macOS Monterey (12.7.6).

> See releases page for downloadable EFI packages: https://github.com/MedoX71T/HP210G1-Hackintosh/releases
